
# REST API Simple mini-marketplace flask

this is mini project build with flask and JWT as a auth and sqlite as a Database


## Run Locally

Clone the project

```bash
  git clone https://github.com/sabiqazhar/simple-market-place.git
```

Go to the project directory

```bash
  cd simple-market-place
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  flask run
```

Documentation

```bash
  http://127.0.0.1:5000/swagger-ui
```
